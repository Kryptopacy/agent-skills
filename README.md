# agent-skills

A collection of agent skills for the [skills.sh](https://skills.sh) ecosystem.

## Skills

### skills-router

A meta-skill that routes any task to the correct specialist skill. Covers 60+ skills across React, Supabase, Stripe, Solana, AI/LLM, testing, mobile, design, marketing, DevOps, and more.

**Install:**
```bash
npx skills add https://github.com/kryptopacy/agent-skills --skill skills-router
```

**Use:** Load this skill at the start of any session to get routed to the right specialist skill for your task — without needing to know skill names yourself.
